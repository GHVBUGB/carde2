🔧 技术实现

主要的技术修改：

1. 去掉标题

// 去掉的代码 <div className="text-center"> <h2 className="text-lg font-semibold">文字模块编辑器</h2> <p className="text-sm text-gray-600">编辑名片上的文字内容和样式</p> </div>

2. 头衔改为选择项

// 修改前：输入框 <Input value={textModules.title} onChange={(e) => handleTextChange('title', e.target.value)} placeholder="请输入您的职位" /> // 修改后：选择框 <select value={textModules.title} onChange={(e) => handleTextChange('title', e.target.value)} > <option value="">请选择职位</option> <option value="首席成长伙伴">首席成长伙伴</option> <option value="金牌成长顾问">金牌成长顾问</option> <option value="五星服务官">五星服务官</option> <option value="学习领航官">学习领航官</option> </select>

3. 数字输入去掉前导0

// 修改前 onChange={(e) => handleTextChange('studentsServed', Number(e.target.value))} // 修改后 onChange={(e) => { const value = e.target.value.replace(/^0+/, '') || '0' handleTextChange('studentsServed', Number(value)) }}

4. 布局优化

// 修改前：垂直布局 <div className="space-y-4"> <div className="space-y-2 mb-4">...</div> <div className="space-y-2 mb-4">...</div> </div> // 修改后：网格布局 <div className="grid grid-cols-1 md:grid-cols-2 gap-4"> <div className="space-y-2">...</div> <div className="space-y-2">...</div> </div>
