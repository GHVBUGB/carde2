📍 文字模块坐标显示功能

🎯 功能说明

现在您可以在名片预览中显示每个文字模块的x、y坐标，这对于：

  • 精确了解每个模块的当前位置
  • 方便手动调整和记录坐标值
  • 调试和优化布局设计
  • 为其他开发者提供准确的坐标信息

🔧 实现方案

在 src/components/card/draggable-business-card-preview.tsx 中添加了坐标显示功能：

1. 添加状态控制

const [showCoordinates, setShowCoordinates] = useState(false)

2. 修改渲染函数

const renderDraggableText = ( moduleId: keyof TextPositions, text: string, style: TextStyles[keyof TextStyles], position: TextPositions[keyof TextPositions], showCoordinates: boolean = false // 新增参数 ) => { return ( <div> {text} {showCoordinates && ( <div style={{...}}> ({position.x}, {position.y}) </div> )} </div> ) }

3. 添加切换按钮

<button onClick={() => setShowCoordinates(!showCoordinates)} className={`px-3 py-1 rounded text-sm transition-colors ${ showCoordinates ? 'bg-orange-500 text-white hover:bg-orange-600' : 'bg-gray-500 text-white hover:bg-gray-600' }`} > {showCoordinates ? '隐藏坐标' : '显示坐标'} </button>

📋 支持的模块

以下所有文字模块都支持坐标显示：

  • 姓名 - 显示在文字下方
  • 职位头衔 - 显示在文字下方
  • 服务人数 - 显示在徽章下方居中
  • 好评率 - 显示在徽章下方居中
  • 联系电话 - 显示在文字下方
  • 外教筛选标签 - 显示在标签下方居中
  • 学情反馈标签 - 显示在标签下方居中
  • 学习计划标签 - 显示在标签下方居中
  • 学习资源标签 - 显示在标签下方居中

  • 居中对齐，位置统一

🧪 使用方法

使用坐标显示功能非常简单：

  1. 进入编辑页面
  2. 在预览区域上方找到工具栏
  3. 点击 "显示坐标" 按钮
  4. 所有文字模块下方都会显示其坐标
  5. 再次点击 "隐藏坐标" 按钮可关闭显示

💡 使用场景

这个功能特别适用于：

  • 精确调整：当您需要精确调整某个模块位置时
  • 记录坐标：当您想记录当前布局的坐标值
  • 调试布局：当布局出现问题时快速定位
  • 分享设计：向其他开发者提供准确的坐标信息
  • 批量修改：当您需要批量修改多个模块位置时

🔧 技术细节

坐标显示的技术实现：

  • 状态管理：使用 React useState 控制显示状态
  • 条件渲染：通过 showCoordinates 状态控制是否显示
  • 绝对定位：坐标显示在模块下方，不影响原布局
  • 响应式设计：坐标显示会根据模块位置自动调整
  • 性能优化：只在需要时渲染坐标显示元素
