## 🚀 部署指南

### 环境要求
- Node.js 18+
- npm/yarn
- Supabase 账号
- Remove.bg API 密钥

### 快速启动
```bash
# 1. 克隆项目
git clone <repository-url>
cd card2

# 2. 安装依赖
npm install

# 3. 配置环境变量
cp env.example .env.local
# 编辑 .env.local 配置必要参数

# 4. 初始化数据库
# 在 Supabase 中执行 src/lib/supabase/schema.sql

# 5. 启动开发服务器
npm run dev
# 或使用启动脚本
./start.sh  # Linux/Mac
start.bat   # Windows
```

### 生产部署
```bash
# Vercel 部署
npm install -g vercel
vercel --prod

# Docker 部署
docker build -t 51talk-card .
docker run -p 3000:3000 51talk-card
```
