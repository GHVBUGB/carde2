📋 测试链接

请依次点击以下链接，观察页面跳转情况：

🏠 主页 📊 /admin (重定向页面) 🔐 /admin/login 📈 /admin/dashboard 🚪 /admin-portal

🧪 调试步骤

1. 检查主页管理员按钮

  1. 访问 主页
  2. 点击导航栏的"管理员"按钮
  3. 观察是否跳转到登录页面

2. 检查管理员专区

  1. 在主页向下滚动找到"管理员专区"
  2. 点击"🔐 管理员登录"按钮
  3. 点击"📊 管理员门户"按钮

3. 手动测试路由

  1. 直接访问 /admin → 应该跳转到登录页
  2. 直接访问 /admin/login → 显示登录表单
  3. 直接访问 /admin/dashboard → 需要权限验证

🔧 可能的问题

1. JavaScript 错误

打开浏览器开发者工具 (F12)，查看 Console 标签页是否有错误信息。

应该看到类似这样的调试信息：

Admin redirect: isLoggedIn = null
Redirecting to login...

2. 路由冲突

可能存在多个路由文件冲突，检查以下文件：

  • src/app/admin/page.tsx - 重定向逻辑
  • src/app/admin/login/page.tsx - 登录页面
  • src/app/admin/dashboard/page.tsx - 仪表板

3. 服务器编译问题

检查终端是否有编译错误或警告信息。

如果有问题，尝试重启开发服务器：

Ctrl + C (停止服务器)
npm run dev (重新启动)
