🔍 技术细节

重叠问题的原因

DraggableBusinessCardPreview 组件使用了两种背景显示方式：

  • CSS背景： 用于日常预览，响应快但导出时可能有兼容性问题
  • IMG元素： 用于导出时确保图片完整渲染

修复方案

确保在不同模式下只显示相应的背景：

  • 预览模式： useImgBackground = false → 只显示CSS背景
  • 导出模式： useImgBackground = true → 只显示IMG元素
