📝 修改初始位置的方法

要修改初始位置，您需要编辑以下文件：

src/store/card.ts

在该文件中找到 initialTextPositions 常量，然后修改相应的坐标值：

const initialTextPositions = { companyName: { x: 16, y: 16 }, name: { x: 175, y: 176 }, title: { x: 175, y: 200 }, studentsServed: { x: 135, y: 288 }, positiveRating: { x: 195, y: 288 }, phone: { x: 175, y: 460 }, teacherSelectionLabel: { x: 40, y: 400 }, progressFeedbackLabel: { x: 120, y: 400 }, planningLabel: { x: 200, y: 400 }, resourceSharingLabel: { x: 280, y: 400 } }

🔧 修改步骤

  1. 打开文件：在您的代码编辑器中打开 src/store/card.ts
  2. 找到位置：搜索 initialTextPositions（大约在第148行）
  3. 修改坐标：将您想要的坐标值替换到对应的 x 和 y 属性中
  4. 保存文件：保存修改后的文件
  5. 重新加载：刷新浏览器页面，新的初始位置就会生效

💡 修改示例

例如，如果您想将姓名位置向右移动50像素，向下移动20像素：

// 修改前 name: { x: 175, y: 176 }, // 修改后 name: { x: 225, y: 196 },
⚠️ 注意事项：
  • 坐标值以像素为单位
  • X坐标：从左到右，0为最左边
  • Y坐标：从上到下，0为最上边
  • 修改后需要刷新页面才能看到效果
  • 建议先备份原始值，以便需要时恢复
